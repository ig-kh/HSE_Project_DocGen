# Рецепт и ттюнинг модели

## Модель
Предлагается использовать небольшой QWEN (8b возможно достаточен) для генеративного NER или перевода - генерировать строгие JSON из неформальных сообщений пользователей.

### Промптинг
Подразумевается провести итерации промпт-инжиниринга (сначала проверить эффект на zero-shot и few-shot оригинальной модели) для формулировки задачи для модели.

### Peft SFT
Модель дотюнивается в эффективной манере с помощью LoRA-адаптера на supervised данных - парах <текст сообщения, JSON в схеме базы данных отправки>.

## Построцессинг
Для того, чтобы выходы модели точно ложились на систему триггеров, JSON-объекты должны контролируемо генерироваться формальной грамматикой посредством GBNF пост процессора.
